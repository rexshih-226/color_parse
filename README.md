衣物去背:
在進行顏色分析前，系統會先將背景移除，只保留衣物本體，避免背景顏色影響辨識結果。

顏色辨識:
針對衣物區域分析主要顏色，並依照顏色比例輸出最多三種代表色。
回傳
{
  "color": "黑色",
  "percent": 72.5
}
系統會依據顏色分布自動調整分析方式，提高辨識效果。

顏色名稱轉換:
分析出的 RGB 顏色會進一步對應到使用者較容易理解的顏色名稱，例如：
白色, 黑色, 灰色, 紅色, 黃色...

處理流程:
圖片輸入
    ↓
衣物偵測 / 去背
    ↓
顏色分析
    ↓
顏色分類
    ↓
輸出主要顏色與比例
